操作步骤

一、打开终端

二、到本地文件目录

cd /home/developer/public_html

三、配置git

用户名：

git config --global user.name "叶绍亭"

邮箱：

git config --global user.email "yeshaoting@163.com"

配色方案：

git config --global color.ui auto
